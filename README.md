# Nova Video Lab

Premium UI demo for text-to-video, image-to-video, and text-to-image. Runs on low-end PCs by using any external API.

## Local
1. `npm install`
2. `Copy-Item .env.example .env`
3. Fill `.env` with your provider info
4. `npm run dev`

## Railway
- Push to GitHub
- Create Railway project from repo
- Add env vars from `.env`
- Deploy

## GitHub Pages (UI only)
To show the premium UI on GitHub Pages:
1. Run `npm run build:docs`
2. GitHub Settings → Pages → Source: `docs/`
